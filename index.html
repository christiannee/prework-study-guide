Prework Study Guide

HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML is used to create the structure of a webpage. It is made up of elements which work together to provide content and the basic framework for all pages on the web.

CSS

  • CSS gives the content style.
  • It defines the color, aesthetics, visual layout, and organization.

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git pull origin main: double checks if our local branch is in sync with the base branch in GitHub.

JavaScript

  • JavaScript is a high-level dynamic programming language that allows users to interact with websites.
