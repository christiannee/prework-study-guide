# Prework Study Guide Webpage

## Description

This webpage is a study guide for students in the coding bootcamp designed to assist them throughout their studies. It consists of key topics and notes over what students will learn in the coding bootcamp, including notes on HTML, CSS, Git, and JavaScript.

## Installation

N/A

## Usage

This study guide consists of different sections of topics learned throughout the bootcamp. These topics include HTML, CSS, Git, and JavaScript. Each section contains notes on each topic. Additionally, if you open the DevTools console, you can see the JavaScript running. This will list topics students learned during the prework section of the bootcamp, and it will also choose one topic for students to study first.
